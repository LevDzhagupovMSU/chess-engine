Учебный шахматный движок, написанный на C++.
Проект реализует базовую логику игры в шахматы и простой искусственный интеллект для выбора ходов. Проект уже содержит готовую сборку. В main.cpp 
реализована игра только за белых на данный момент, однако это элементарно исправляется.

Возможности
1)Полная базовая логика шахмат:
    генерация допустимых ходов
    проверка на шах и мат, проверка на пат
2) Простой искусственный интеллект
3) Поиск лучшего хода с настраиваемой глубиной
4) Возможность играть против компьютера

В методе getAiMove есть парамерт глубины, который показывает насколько далеко ИИ будет оценивать позицию и находить наилучший вариант продолжения.
Таким образом можно настроить желаемый уровень ИИ.

Проект создан в учебных целях и предназначен для:
  понимание принципов объектно-ориентированного программирования
  понимания работы шахматных движков
  практики программирования на C++
